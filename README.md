# 🐪 **3D Rasterizer**

### **Prerequisites**
---
* CMake >=3.21
* Ninja
* SDL2
* SDL2_Image
* SDL2_TTF
* C++17

### **How to build**
---

1. Add SDL32 or SDL64 to libraries/

```sh
cd ./build
cmake -G "Ninja" ..
cd ..
./builandrun.bat
```
